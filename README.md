# Salah-Project
Salah Time is a Node.js based application designed to help users view daily prayer times in an easy and reliable way. The system calculates and displays Salah timings according to the user’s locationThe project is developed using the Model–View–Controller (MVC) architecture, which separates application logic, user interface, and data handling.Salah Time focuses on clean structure and practical implementation, making it suitable for learning and academic project purposes.
# Features
1. Displays daily Salah (prayer) times
2. Location-based prayer time support
3. Responsive frontend using React
4. Well-structured MVC backend
# Technologies Used
1. Node.js
2. React
